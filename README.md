# IntelliStock

A Smart Inventory & Sales Forecasting System built using Java, SQLite, JDBC, Multithreading and Python Analytics.

## Features

- Inventory Management
- SQLite Database Integration
- JDBC Repository Layer
- Strategy Pattern Pricing
- Inventory Simulation
- Automatic Restocking
- Sales Recording
- Python Analytics Dashboard
- Multithreading

## Technologies

- Java
- JDBC
- SQLite
- Python
- Pandas
- Matplotlib

## Project Structure

src/
- products/
- pricing/
- jdbc/
- simulation/
- interfaces/
- exception/

## Running

Compile:

```bash
javac ...
```

Run:

```bash
java -cp "bin;lib/*" simulation.SimulationTest
```
